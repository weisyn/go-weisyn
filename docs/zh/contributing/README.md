## 🎯 贡献流程

### 1. 准备开发环境

阅读 [开发环境搭建](./development-setup.md)，准备本地开发环境。

### 2. 了解项目结构

阅读 [设计文档说明](./design-docs.md)，了解项目的设计文档和代码结构。

### 3. 遵循编码规范

阅读 [代码规范](./code-style.md) 和 [文档规范](./docs-style.md)，确保你的贡献符合项目标准。

### 4. 提交 Pull Request

1. Fork 项目仓库
2. 创建特性分支
3. 提交你的更改
4. 创建 Pull Request
