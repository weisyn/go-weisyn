## 🎯 按症状排查

### 节点问题

- 节点无法启动
- 节点同步缓慢
- 节点频繁重启

### 网络问题

- 无法连接到其他节点
- 网络延迟过高
- 消息丢失

### 性能问题

- 交易处理缓慢
- 内存使用过高
- 磁盘空间不足
