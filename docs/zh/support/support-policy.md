# 支持策略

---

## 概述

本文档定义了 WES 版本的支持策略和生命周期。

---

## 版本类型

| 类型 | 说明 | 支持期限 |
|------|------|----------|
| LTS | 长期支持版本 | 2 年 |
| Current | 当前版本 | 至下一版本发布 |
| Preview | 预览版本 | 不提供支持 |

---

## 支持级别

### 完整支持

- Bug 修复
- 安全更新
- 性能改进
- 文档更新

### 安全支持

- 仅安全更新
- 关键 Bug 修复

### 无支持

- 不再接收任何更新
- 建议升级

---

## 版本生命周期

```
发布 → 完整支持 → 安全支持 → EOL（生命周期结束）
```

### LTS 版本时间线

| 阶段 | 时长 |
|------|------|
| 完整支持 | 12 个月 |
| 安全支持 | 12 个月 |
| 总计 | 24 个月 |

### Current 版本时间线

| 阶段 | 时长 |
|------|------|
| 完整支持 | 至下一版本发布 |
| 安全支持 | 3 个月 |

---

## 升级建议

### 何时升级

- 新的 LTS 版本发布时
- 当前版本进入安全支持期
- 需要新功能时

### 升级路径

- 支持跨一个主版本升级
- 大版本升级需要参考迁移指南
- 建议先在测试环境验证

---

## 安全响应

### 安全漏洞等级

| 等级 | 响应时间 |
|------|----------|
| 严重 | 24 小时内 |
| 高 | 72 小时内 |
| 中 | 7 天内 |
| 低 | 下一版本 |

### 安全更新通知

- 邮件列表
- GitHub Security Advisories
- 官方博客
