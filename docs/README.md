## ğŸ“– About WES

WES is a **third-generation blockchain** that introduces the **ISPC (Intrinsic Self-Proving Computing)** paradigm, breaking through the deterministic consensus limitations of traditional blockchains.

### Core Innovations

- **ISPC Verifiable Computing**: Single execution + multi-point verification, supporting AI and complex computations on-chain
- **EUTXO Extended Ledger**: Three-layer output architecture (Asset/Resource/State) with strong parallel processing capability
- **URES Unified Resource Management**: Content-addressed storage for unified management of contracts/AI models/files
- **PoW+XOR Distance Selection Consensus**: Proof of Work + XOR distance selection for high-performance consensus

- âœ… **AI Native**: The only blockchain supporting on-chain AI model inference
